# Anonymous VC IP Extractor

## Deployed on Railway

### Commands
- `/start` - Show help
- `/scan` - Scan active voice chats
- `/extract` - Extract IPs
- `/status` - Check status
- `/stop` - Stop bot

### Environment Variables
- API_ID: Telegram API ID
- API_HASH: Telegram API Hash
- SESSION_STRING: Pyrogram session
- BOT_TOKEN: Telegram bot token
- ADMIN_ID: Your user ID
- AUTO_MODE: Enable auto scan
- SCAN_INTERVAL: Auto scan interval
